Greeting RESTful Service

REST Endpoints

DefaultGreetingResource

GET greeting count
GET all greetings
GET all greeting objects as JSON. Pass the request with Accept: application/json
GET all greeting objects as XML. Pass the request with Accept: application/xml
GET all greeting objects as JSON or XML depending on request's Accept header. Pass the request with Accept: application/json or Accept: application/xml GET all languages
GET the greeting for given language such as resources/greetings/turkish
POST a new greeting using language and greeting such as resources/defaultGreetings/arabic/assalamualaikum.
